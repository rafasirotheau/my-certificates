# my-certificates

Repository of certification programs completed by me.

## Repository Structure

This repository is organized by certificate folders, where each folder name is a "slugified" version of the issuer + program name.

### Folder Naming Convention

Format: `issuer-program-name` (lowercase, hyphen-separated)

Example:
- **Issuer**: IFTL
- **Program**: Tech Lead Program
- **Folder name**: `iftl-tech-lead`

### Folder Contents

Each certificate folder contains:
- **Certificate file**: Usually a PDF document
- **README.md**: Summary of the program and additional information
